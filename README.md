# ちょこパス
KeePassに替わる、モダンでセキュアなデスクトップパスワードマネージャー。

## ✨ 機能
- **🔐 KeePass CSV インポート**: KeePass の CSV エクスポートからデータをインポート
- **💾 パスワード管理**: インポートしたパスワードの表示、コピー、管理
- **🛡️ セキュリティ重視**: すべてのデータをローカルに保存、外部接続なし

## 🛠️ 技術スタック

| 技術             | 用途                                          |
| ---------------- | --------------------------------------------- |
| **Electron**     | デスクトップアプリケーションフレームワーク    |
| **TypeScript**   | 型安全な JavaScript                           |
| **React 19**     | UI フレームワーク                             |
| **Shadcn UI**    | コンポーネントライブラリ                      |
| **Tailwind CSS** | ユーティリティファーストな CSS フレームワーク |
| **Lucide Icons** | アイコンライブラリ                            |

## 📋 CSV インポート形式

### 対応するカラムヘッダー

- **Title** / Name（タイトル）
- **Username** / Login（ユーザー名）
- **Password**（パスワード）
- **URL** / Url（URL）
- **Notes**（メモ）
- **Group**（グループ）

## 🔒 セキュリティについて
- すべてのデータはお使いのマシンにローカルで保存されます
- データは外部サーバーに送信されません
